## 1. 문제 설명

N명의 사람이 있을 때 특정 A, B가 서로 만나 승부할 라운드를 구해라

## 4. 올바른 접근 및 풀이

N 값이 무조건 2의 n승.
A, B를 2로 나누고 올림함 값이 같을 때 서로 승부했다고 볼 수 있음
그래서 두 값이 같아질 때까지 2로 나누고 올림

```js
function solution(n, a, b) {
  let answer = 0;

  while (a != b) {
    a = Math.ceil(a / 2);
    b = Math.ceil(b / 2);
    answer += 1;
  }

  return answer;
}

console.log(solution(8, 4, 7));
console.log(solution(8, 1, 2));
```
